# SoftDesk 生产就绪度评估报告

> 评估方式：逐模块代码级核查，对照 PRD 与技术架构文档逐文件阅读源码。
> 评判标准：真实投入生产上线（Production-ready）。
> 生成日期：2026-06-27

---

## 总体结论

**当前状态：功能演示成熟（Demo-ready），但距离生产上线还有明显差距（Not production-ready）。**

代码工程质量整体偏高——有缓存、并发控制、安全白名单、事务、降级处理、单元测试。但存在 **3 个会直接阻断上线的严重问题（P0）**，以及一批"宣传≠实现"和"假开关"的信任风险。最致命的是：**实际行为与 PRD/UI 的隐私承诺正面冲突**，以及**全平台只支持 macOS**。

| 优先级 | 数量 | 含义 |
|--------|------|------|
| P0 阻断上线 | 3 | 不修复不能上线 |
| P1 信任风险/假开关 | 5 | 影响用户信任与数据准确性 |
| P2 半成品/功能落差 | 5 | 体验缺陷或宣传落差 |

## 一、按模块的功能完成度

### 模块 1：软件扫描与启动 — 🟢 完成（仅 macOS）

| 功能 | 状态 | 说明 |
|------|------|------|
| 软件扫描 | ✅ 完成 | 三目录扫描、plist 解析、三级图标回退、内存+磁盘双缓存、并发限流、安装监听自愈 |

**参考**：`electron/scanner.ts:150-154`、`electron/main.ts:495-497`

## 二、阻断上线的三大严重问题（P0）

### P0-1：隐私承诺与实现正面冲突（合规/欺诈风险）

文档和 UI 宣称"数据完全不同步"，实际同步了软件清单、使用行为、配置。

**修复方向（二选一，不能两面）**：
- 删除云同步逻辑，使代码符合"数据完全不同步"承诺；或
- 修订 PRD/UI 文案如实披露同步范围，并提供用户可控的同步开关。

---

### P0-2：Supabase RLS 水平越权漏洞（IDOR）

`favorites`/`favorite_groups`/`workflows` 三表的 RLS 用了**从未被设置的** `current_setting('app.current_user_id')`，且 `user_id` 是客户端可控的 text 字段。**任何持 anon key 的人可读写他人数据**。anon key 又被编译进客户端可提取，构成可实际利用的泄露链。

**修复方向**：三表 `user_id` 改为 `uuid references auth.users(id)`，RLS 统一用 `auth.uid()`（与 `ai_configs`/`radial_configs` 对齐）。

**参考**：`SUPABASE_SETUP.md:44-45`

---

### P0-3：全平台仅支持 macOS

扫描、启动、监控三大核心全部硬绑 macOS 系统命令，Windows 上静默失效无任何提示。PRD 承诺"Win/macOS 双平台"未兑现。

**修复方向**：要么砍掉 Windows 宣传，要么补齐 Windows 适配（注册表/开始菜单扫描、`GetForegroundWindow` 监控等）。

## 三、信任风险 / 假开关（P1）

| 问题 | 说明 | 修复方向 |
|------|------|---------|
| Token 写入 localStorage | `supabase-js` 默认 `persistSession:true`，绕过"不进 localStorage"约束 | `createClient` 加 `auth:{persistSession:false}` |
| `scanOnStartup` 开关失效 | Layout 无条件扫描，开关形同虚设 | 启动副作用读取该开关 |
| 5 个假设置 | `launchNotifications`/`weeklyReport`/`autoUpdates`/`anonymizeData`/`sendAnalytics` 无消费方；其中后两者是隐私承诺却无代码兑现 | 实现或移除，尤其隐私类 |
| 使用时长无空闲检测 | 数据准确性硬伤 | 接入 `powerMonitor.getSystemIdleTime` |
| 统计时区不一致 | 跨午夜数据错位 | 统一日期口径为本地时区 |

---

## 四、半成品 / 功能落差（P2）

1. **智能推荐 `frequentPairs` 恒为空数组**，共现画像维度形同虚设（DB 有 `getCoUsage` 却没接）；且无模型门控，刷新可能空转无提示。
2. **智能推荐只能重排已安装软件**，无法发现新软件，与"推荐"直觉有落差。
3. **月度报告**完全缺失。
4. anthropic/gemini **测试连接必失败**。
5. 工作流"编排"实为批量并发启动。
