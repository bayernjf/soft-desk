# Supabase 收藏同步配置

## 1. 创建项目

访问 https://supabase.com，创建一个新项目。

## 2. 获取 API 凭证

进入 Project Settings → API：
- `Project URL` → 填入 `.env` 的 `VITE_SUPABASE_URL`
- `anon public` → 填入 `.env` 的 `VITE_SUPABASE_ANON_KEY`

## 3. 创建 favorites 表

在 Supabase SQL Editor 中执行：

```sql
-- 收藏表：跨设备同步用户收藏的软件
create table if not exists favorites (
  id bigint generated by default as identity primary key,
  user_id text not null,
  software_id text not null,
  name text not null,
  bundle_id text,
  category text,
  icon text,
  color text,
  created_at timestamptz default now(),
  unique(user_id, software_id)
);

-- 索引：加速查询
create index if not exists idx_favorites_user_id on favorites(user_id);

-- 启用 RLS
alter table favorites enable row level security;

-- 策略：用户只能操作自己的收藏（user_id 由应用传入）
create policy "用户可读写自己的收藏"
  on favorites
  for all
  using (user_id = current_setting('app.current_user_id', true))
  with check (user_id = current_setting('app.current_user_id', true));
```

## 4. 创建 ai_configs 表（AI 配置跨设备同步）

在 Supabase SQL Editor 中执行：

```sql
-- AI 配置表：跨设备同步非敏感字段（不含 apiKey）
create table if not exists ai_configs (
  id bigint generated by default as identity primary key,
  user_id uuid references auth.users(id) not null,
  provider text not null,
  endpoint text,
  model text not null,
  name text not null,
  is_active boolean default false,
  created_at timestamptz default now()
);

-- 索引：加速查询
create index if not exists idx_ai_configs_user_id on ai_configs(user_id);

-- 启用 RLS
alter table ai_configs enable row level security;

-- 策略：用户只能操作自己的配置
create policy "用户可读写自己的 AI 配置"
  on ai_configs
  for all
  using (user_id = auth.uid())
  with check (user_id = auth.uid());
```

## 5. 创建 workflows 表（工作流跨设备同步）

在 Supabase SQL Editor 中执行：

```sql
-- 工作流表：跨设备同步用户创建的工作流
create table if not exists workflows (
  id bigint generated by default as identity primary key,
  user_id text not null,
  workflow_id text not null,
  name text not null,
  description text not null,
  software_ids text[] not null,
  usage_count integer default 0,
  last_used timestamptz default now(),
  is_favorite boolean default false,
  color text not null,
  updated_at timestamptz default now(),
  unique(user_id, workflow_id)
);

-- 索引：加速查询
create index if not exists idx_workflows_user_id on workflows(user_id);

-- 启用 RLS
alter table workflows enable row level security;

-- 策略：用户只能操作自己的工作流（user_id 由应用传入）
create policy "用户可读写自己的工作流"
  on workflows
  for all
  using (user_id = current_setting('app.current_user_id', true))
  with check (user_id = current_setting('app.current_user_id', true));
```

## 6. 配置 .env

```bash
cp .env.example .env
# 编辑 .env，填入你的 Supabase URL 和 Anon Key
```

## 6. 验证

登录 SoftDesk 后：
- 收藏任意软件，在 Supabase Table Editor 中查看 favorites 表
- 添加 AI Provider，在 ai_configs 表中查看非敏感字段是否已同步
